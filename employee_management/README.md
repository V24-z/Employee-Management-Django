# Employee Management System

A Django-based web application to manage employee records with CRUD operations,
search, filters, pagination, and image upload.

## Features
- Add, update, delete employees
- Search and filter by name, department, role
- Pagination (10 records per page)
- Image upload
- Bootstrap UI

## Tech Stack
- Python
- Django
- SQLite
- HTML, CSS, Bootstrap

## How to Run
1. Clone the repository
2. Create virtual environment
3. Install requirements
4. Run `python manage.py runserver`
